About - Concept/Module Used In Angular App

This app is built from scratch to learn angular

Used concept are -

  • Angular Installation
  • Folder Structure Understanding
  • Generate Component
  • Use Of Component & HTMl File along with Css
  • Form Builder
  • Template Driven Form
  • Reactive Form Validation
  • Event Emitter
  • Input Output Decorator
  • ViewChild Decorator - passing data back to child
  • Local Storage
  • Toaster
  • Bootstrap & JQuery with Angular
  • Use of ANgular Hooks - ngOnInit, ngDestroy, afterViewInit
  • Global/Common component - Toast which can use in any component and service
  • Material UI
  • Custom Common Services
  • Http Services
  • Interceptor Service
  • Routing with Auth Guard
  • Routing with Parent and child route
  • Lazy Loading Of Modules In Routing
  • Dynamically change active class for route menu
  • Hide/Show Route based on Authentication
  • Custom Module - Component export, provide, declare
  • Injectable & Service Injection
  • Data Table Using Material UI
  • Data Table Using Bootstrap/JQuery
  • Pagination, Scroll Loading
  • Shared Service - Data Passing Between Non-Relational Component
  • Data Passing Using Input Output Decorators In Relational Component
  • Input Decorators with getter & Setter Method
  • RX JS - Observables, Operators etc
  • Custom Directives - Attribute & Structural Directives
